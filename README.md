# sistema-de-divulga-o-de-e-sports

# Portal de Divulgação de Campeonatos (Mobile & Console)

# Descrição do Projeto
Plataforma centralizada para que organizadores possam divulgar torneios de e-sports, permitindo que jogadores busquem competições por filtros de jogo e plataforma.

# Objetivos do Sistema
- Facilitar a visibilidade de torneios amadores e semiprofissionais.
- Oferecer filtros de busca por plataforma (Console, Mobile e PC).
- Criar uma vitrine organizada para a comunidade gamer.

# tecnologias e ferramentas Propostas
- Banco de Dados: MySQL
- Frontend: HTML, CSS e JavaScript
- Linguagem Principal: Java (Spring Boot)
- Interface: HTML, CSS e JavaScript
- Gestão de Projeto: Trello (Metodologia Ágil)
- Design: Figma (Prototipagem de UI/UX) 
- Modelagem de Dados: dbdiagram.io

# Estrutura do Banco de Dados
O banco de dados foi modelado para garantir a integridade dos dados e facilitar a filtragem. As tabelas principais são:
1. Usuarios: Cadastro de organizadores e jogadores.
2. Plataformas: Diferenciação entre Mobile, Console ou PC.
3. Jogos: Listagem de títulos (eFootball, Free Fire, etc.).
4. Campeonatos: Onde as informações se cruzam (tabela principal).
